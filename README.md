# Employee Management System

A Python-based Employee Management System using PostgreSQL and Object-Oriented Programming.

## Features
- Add Employee
- View Employees
- Search Employee
- Update Employee
- Delete Employee
- Department Management
- PostgreSQL Database Integration

## Technologies Used
- Python
- PostgreSQL
- psycopg2
- python-dotenv

## Installation

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file using `.env.example`.
4. Run:

```bash
python main.py
```
